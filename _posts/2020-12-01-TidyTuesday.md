TidyTuesday is a community project for those looking to skill-up on data analysis and visualization using the R programming language. Each week the community moderators publish a random dataset on [Github](https://github.com/rfordatascience/tidytuesday) and participants share their data vizzes and code on Twitter using the hashtag TidyTuesday.
